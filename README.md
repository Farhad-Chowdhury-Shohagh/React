# React Projects

A collection of beginner-friendly React.js mini projects built in a single React folder using **Vite**. This repository is designed for practicing core React concepts such as components, JSX, hooks, state management, event handling, conditional rendering, forms, and responsive UI design.

## Projects Included

1. **Counter App** - Increase, decrease, and reset a number.
2. **To-Do List App** - Add and delete daily tasks.
3. **Calculator App** - Perform basic arithmetic operations.
4. **Digital Clock** - Display live time and current date.
5. **Grade Calculator** - Calculate grades based on marks.
6. **Expense Tracker** - Add expenses and calculate total cost.
7. **Quiz App** - Answer questions and view final score.
8. **Notes App** - Write, save, and delete notes.
9. **Password Generator** - Generate random secure passwords.
10. **Color Picker** - Select and preview colors instantly.

## How to Run the Project

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### Step 2: Go to the Project Folder

```bash
cd React_10_Projects
```

### Step 3: Install Dependencies

```bash
npm install
```

### Step 4: Start the Development Server

```bash
npm run dev
```

### Step 5: Open in Browser

After running the project, Vite will show a local server link like:

```text
http://localhost:5173/
```

Open that link in your browser.

## How to Use

After opening the project in the browser, a sidebar will appear with all 10 project names. Click on any project name to view and use that specific React mini project.
